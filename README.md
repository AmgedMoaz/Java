## 🚀 How to Run
 
1. Clone the repository:
```bash
   git clone https://github.com/AmgedMoaz/Java.git
```
2. Open the project folder in IntelliJ IDEA (or any Java IDE).
3. Run the `main` class of the project you want to try.
Alternatively, from the command line:
```bash
javac Main.java
java Main
```
